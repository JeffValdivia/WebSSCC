# WebSSCC - Juegos Deportivos de Confraternidad 2026

Aplicación web para gestionar y visualizar los Juegos Deportivos de Confraternidad del Colegio de los Sagrados Corazones.

## 🏗️ Arquitectura

El proyecto está dividido en dos directorios principales:

- **`/client`** - Frontend React + Vite
- **`/server`** - Backend Node.js + Express

## 📋 Requisitos

- Node.js 16+
- npm o yarn

## 🚀 Instalación y Uso

### Backend

```bash
cd server
npm install
npm start
```

El servidor se ejecutará en `http://localhost:3001`

### Frontend

```bash
cd client
npm install
npm run dev
```

La aplicación se ejecutará en `http://localhost:5174`

## 🎯 Características

- ✅ Visualización de fixtures (partidos) para tres deportes
- ✅ Posiciones (tabla de posiciones) de todos los torneos
- ✅ Clasificación por género (varones, damas, mixto)
- ✅ Diseño responsive para mobile y desktop
- ✅ Leyenda de colores por promoción

## 🏐 Deportes Disponibles

- ⚽ Fútbol (varones y damas)
- 🏀 Básquet (varones y damas)
- 🏐 Vóley (damas y mixto)

## 📱 Responsividad

La aplicación está optimizada para:
- Móviles (pantallas pequeñas)
- Tablets
- Escritorio
